# molam-id

Brique 1 - Molam ID (Fastify + PostgreSQL)
## Quick start

1. Start DB:
   ```
   docker compose up -d
   ```

2. Install deps:
   ```
   npm install
   ```

3. Run server:
   ```
   npm start
   ```

Adminer: http://localhost:8080
API:
- POST /api/signup
- POST /api/login
- POST /api/refresh
