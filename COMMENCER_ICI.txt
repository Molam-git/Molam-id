┌────────────────────────────────────────────────────────────────┐
│  ÉTAPE 1: ARRÊTER TOUS LES PROCESSUS NODE                     │
└────────────────────────────────────────────────────────────────┘

Option A (RECOMMANDÉE - La plus simple):
   1. Appuyez sur Ctrl + Shift + Esc (Gestionnaire des tâches)
   2. Onglet "Détails"
   3. Trouvez tous les "node.exe"
   4. Clic droit → "Fin de tâche" sur chaque node.exe
   5. Fermez le Gestionnaire des tâches

Option B (Plus rapide):
   Redémarrez simplement votre ordinateur


┌────────────────────────────────────────────────────────────────┐
│  ÉTAPE 2: DÉMARRER LE BACKEND                                 │
└────────────────────────────────────────────────────────────────┘

   1. Ouvrez l'Explorateur de fichiers
   2. Allez dans: C:\Users\lomao\Desktop\Molam\Molam-id
   3. Double-cliquez sur: restart-backend.bat

   ➜ Une fenêtre noire va s'ouvrir et afficher:
     "📡 Server listening on port 3000"
     "✅ Test de connexion réussi"

   ➜ LAISSEZ CETTE FENÊTRE OUVERTE !


┌────────────────────────────────────────────────────────────────┐
│  ÉTAPE 3: DÉMARRER L'INTERFACE WEB                            │
└────────────────────────────────────────────────────────────────┘

   1. Ouvrez l'Explorateur de fichiers
   2. Allez dans: C:\Users\lomao\Desktop\Molam\Molam-id\brique-36-ui-id\web
   3. Double-cliquez sur: start-ui.bat

   ➜ Une deuxième fenêtre noire va s'ouvrir et afficher:
     "VITE ready in XXX ms"
     "Local: http://localhost:5173"

   ➜ LAISSEZ AUSSI CETTE FENÊTRE OUVERTE !


┌────────────────────────────────────────────────────────────────┐
│  ÉTAPE 4: UTILISER LE SYSTÈME                                 │
└────────────────────────────────────────────────────────────────┘

   1. Ouvrez votre navigateur (Chrome, Edge, Firefox...)

   2. Allez à l'adresse: http://localhost:5173

   3. Vous verrez l'interface Molam-ID !
      → Cliquez sur "Créer un compte" pour vous inscrire
      → Ou "Connexion" si vous avez déjà un compte
